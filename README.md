# Health Sense System

Welcome to the official repository for **Health Sense System**.

🌐 **Live Website:** healthsensesystem.com

## About

Health Sense System is a web-based platform focused on health awareness, wellness information, and digital solutions related to healthcare and healthy living.

This repository contains the website source code built using modern web technologies.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` — Main website page
* `styles.css` — Styling and layout
* `script.js` — Interactive functionality
* `images/` — Website assets and media
* `CNAME` — Custom domain configuration

## Features

* Responsive website design
* Fast and lightweight performance
* User-friendly interface
* Cross-device compatibility
* Custom domain integration
